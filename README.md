# ⚔️ 진격의 거인 배틀 (Attack On Titan Battle)

냥코대전쟁 게임 메커니즘을 기반으로 **진격의 거인** 테마를 적용한 타워 디펜스 게임입니다!

## 🎮 게임 방법

### 기본 규칙
1. **골드를 모아** 조사병단 캐릭터를 배포합니다
2. **캐릭터들이 거인들을 격퇴**합니다
3. **10개의 스테이지**를 모두 클리어하면 승리!
4. **조사병단 본부 HP가 0**이 되면 패배

### 조작 방법
- **캐릭터 선택**: 아래 그리드에서 캐릭터 클릭
- **배포**: 선택한 캐릭터 정보 영역의 "배포하기" 버튼 클릭
- **게임 제어**: 
  - 일시정지/계속 버튼으로 게임 일시정지 가능
  - 업그레이드 버튼으로 능력 강화
  - 게임 리셋으로 새 게임 시작

## 💪 업그레이드 시스템

### 🧬 캐릭터 강화 (100 G)
- 모든 캐릭터 레벨 +1
- HP와 공격력 20% 증가

### 💰 자금 증가 (80 G)
- 드롭되는 자금 +10%
- 여러 번 구매 가능

### ❤️ HP 증가 (120 G)
- 조사병단 본부 최대 HP +20
- 여러 번 구매 가능

### ⚔️ 공격력 증가 (100 G)
- 모든 캐릭터 공격력 +10%
- 여러 번 구매 가능

| 스테이지 | 적의 수 | 난이도 |
|----------|--------|--------|
| 1-2 | 3-4 | 쉬움 |

| 5-6 | 7-8 | 어려움 |
| 7-8 | 5-6 | 매우 어려움 |
| 9-10 | 7-8 | 지옥 |

## ⚙️ 난이도 설정

- **쉬움**: 적의 체력 20% 감소
- **중간**: 표준 난이도

## 🎯 게임 팁

1. **조기 업그레이드**: 초반에는 자금을 모아 업그레이드에 투자하세요
2. **캐릭터 조합**: 다양한 캐릭터를 섞어서 사용하면 효율적입니다
3. **난이도 선택**: 첫 플레이는 쉬움 난이도에서 시작하세요
4. **골드 관리**: 필요한 만큼만 배포하고 나머지는 업그레이드에 사용하세요
